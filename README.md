# Research Papers Implementation from Scratch

This repository contains implementations of various research papers from scratch using PyTorch.

## 📚 Implemented Papers

### 1. [LoRA: Low-Rank Adaptation of Large Language Models](./LoRA-Low-Rank-Adaptation/)
- **Paper**: [LoRA: Low-Rank Adaptation of Large Language Models](https://arxiv.org/abs/2106.09685)
- **Authors**: Edward J. Hu, Yelong Shen, et al. (Microsoft)
- **Implementation**: Complete LoRA implementation with MNIST fine-tuning example
- **Status**: ✅ Complete
